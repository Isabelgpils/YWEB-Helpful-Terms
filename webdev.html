Web Development

  • Waterfall: A project management structure that was used prior to Agile in many sectors. It is a more linear and comprehensive management structure.

  • Agile: A project management paradigm that is used widely in the tech industry. It uses three week cycles (sprints) and allows adaptability if things change.

  • Frontend Web Development: What is displayed and is visible to users. Includes HTML, CSS, Javascript.

  • Backend Web Development: The data that is not visible to the user that supports the functionality of a website. Some programming languages: Python, Ruby, Java, Node,PHP.

  • Full-stack development: Includes both frontend and backend web development.

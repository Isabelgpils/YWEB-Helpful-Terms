HTML
Hypertext Mark-up Language		 The computer language that is the basic building block of all the web.
CSS
Cascading Style Sheets		 Computer language that adds styling to web pages.
  • Inline CSS - CSS styling within the html element (within the carets). The styling will affect only that html element.
  • Internal CSS - CSS styling within the head section of the html page. Enclosed in a style tag. The styling will affect the entire page.
  • External CSS - An external CSS page that can be linked within the head section of the html page using a link. The styling will affect any html pages that link to external css page.
Javascript Computer language that provides interactivity and functionality to web pages.
